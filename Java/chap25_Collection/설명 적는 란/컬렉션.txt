Collection
	데이터를 저장, 관리 및 조작하기 위한 인터페이스와 클래스의 모음을 나타냄
	컬렉션 클래스를 데이터를 구조화 하고 다양한 방법으로 접근 조작 할 수 있는 다양한 자료구조를 제공
	자주 사용하는 컬렉션 인터페이스 및 클래스
		List
		set
		map
	메소드 
	sort() : 오름차순
	reverse() : 내림차순
	remove() : 지정된 요소를 컬렉션에서 제거
	isEmpty() : 컬렉션이 비어 있는지 여부 확인
	addAll() : 다른 컬렉션의 모든 요소를 현재 컬렉션에 추가
	contains() : 컬렉션이 지정된 요소를 포함하고 있는지 여부
	clear() : 모든 요소 제거
	toAraray() : 컬렉션의 요소를 배열로 반환
	iterator() : 컬렉션 요소에 반복하는데 사용되는 반복자를 반환
