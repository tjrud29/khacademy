Iterator
	컬렉션 요소를 순회하고 각 요소에 접근하는데 사용되는 인터페이스
	List, set ,map등과 같은 컬렉션을 순차적으로 처리할 때 유용
	
	iterator() : 이 메서드는 초기화
	hasNext() : 이 메서드는 iterator가 순회할 다음 요소가 있는지 검사
				다음 요소가 있으면 'true' 를 반환
				더 이상 순회할 요소가 없으면 false
				
	next() : 이 메서드는 다음 요소로 이동, 해당 요소를 반환
			이 메서드를 호출하기 전에 hashNext()를 사용하여 
			요소가 있는지 확인
			요소가 없는 상태에서 next()를 호출하면 예외가 발생할 수 있음
			
	remove() : 현재 요소 삭제
